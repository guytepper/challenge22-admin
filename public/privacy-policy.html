Challenge 22 Privacy Policy

The Challenge 22 Admin Dashboard provides tools for the challenge 22 administrators.

In order to do that, the website uses Facebook as the login provider.

We request access your facebook email, public profile and user friends, which is required for using facebook login, but we also ask for persission to get your managed groups & photos, in order to be able to publish album photos to the Challenge 22 facebook groups.

We won't use these information we get from you for other purposes.
